# GrokFS App Icon Prompt

Create a production-ready iOS app icon for an app named GrokFS, a Grok-powered mobile coding workspace with a persistent Unix-like filesystem.

Square 1024 by 1024 app-icon artwork. Fully opaque. No transparency. Do not bake in an iOS rounded-corner mask. No text, letters, logos, faces, mascots, or trademarked symbols.

Use one memorable central geometric mark: a precise terminal prompt chevron integrated with three stacked filesystem layers. The chevron should imply forward execution; the layers should imply a durable workspace. The combined mark must remain unmistakable at 29 pixels.

Style: intelligent, technical, calm, premium, and slightly futuristic. Flat vector-like geometry with subtle material depth, sharp alignment, generous safe margins, and a strong silhouette. Avoid visual noise, tiny details, gradients, glow, glass effects, and decorative shadows.

Palette: near-black graphite background, crisp off-white primary symbol, restrained teal status accent, and a small warm-amber cursor accent. Centered, optically balanced, Apple-quality composition.
